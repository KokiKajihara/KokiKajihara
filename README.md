## ⚡ About

> AIを **「使う」** のではなく **「組む」** ことで、  
> ソロでも複数領域の創作を成立させる働き方を探求しています。  
> 音楽生成AI・LLM・自動化ツールを組み合わせ、音楽から映像、Web、執筆までを一気通貫で制作。

| | |
|---|---|
| 🌏 **拠点** | Japan |
| 🎵 **ブランド** | NovaChord（AI音楽プロジェクト） |
| 🔭 **探求領域** | AI音楽生成 · 映像自動化 · LLMO/AIO · ソロ起業 |
| 🤝 **スタンス** | Human × AI の対等なコラボレーション |

## 🎛️ Activities

<details open>
<summary>🎵 &nbsp;<strong>AI Music Creation — NovaChord</strong></summary>
<br>

SUNOをはじめとするAI音楽生成ツールを駆使した楽曲制作。  
**プロンプトエンジニアリング × 音楽理論** を組み合わせ、AIとの共作という形で楽曲を発表しています。

</details>

<details open>
<summary>🎬 &nbsp;<strong>Music Video Automation</strong></summary>
<br>

音声解析（**librosa**）→ AI映像生成（**Runway**）→ 編集合成（**MoviePy / FFmpeg**）までを自動化する個人パイプラインを開発中。

</details>

<details open>
<summary>🌐 &nbsp;<strong>LLMO / AIO Research</strong></summary>
<br>

AIに発見・引用される情報設計を研究。SEO以後のWebコンテンツ戦略として、**構造化データ × AI可読な記述設計** を実践しています。

</details>
